Quiz Game

This C program is a simple quiz game where the user is asked ten questions with multiple-choice answers. After answering each question, the program provides feedback on whether the answer was correct or incorrect and assigns points accordingly. At the end of the game, the total score is displayed. The user can choose to start or end the game.

BTS Information Website

This HTML file presents information about the BTS band, including details about its members and some of their top music videos and favorite songs. It includes images of the band members along with their descriptions and also embeds videos and audio players for their songs. The CSS file contains styles to enhance the visual presentation of the content, including font sizes, colors, and layout adjustments.

Netflix Clone

This is just a clone of Netflix Website

HTML Description:

The HTML structure consists of different sections: A navigation bar with the Netflix logo and buttons for language selection and sign-in. A hero section with a background image and text content encouraging users to join. Several sections showcasing features of Netflix, each with an image and text. A frequently asked questions (FAQ) section with expandable boxes for each question. A footer section with contact information and and links.


CSS Description:

The CSS defines styles for various elements, including fonts, colors, alignments, and layout. It sets background images, sizes, and positions for different sections. Styles for buttons, inputs, and other interactive elements are defined to match the Netflix theme. Media queries are used to ensure responsiveness across different screen sizes
